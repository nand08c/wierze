# wierze
A Peer to Peer Versioning Control System

## Description
"wierze" comes is the phrase for "i believe" in english. This program tries to privatize the development control with the people you choose to believe.

## Features

* Author-Side Integration Control
* Limited to Merging in Author Repository, no branching
* Full History Integrity Protection
* Reversing of History and Mandatory Propagation to Peers
